# 暖通计算工作台原型

本目录是需求阶段的交互参考，不是生产代码。原型使用静态 HTML/CSS/JavaScript，核心示例数值取自当前权威 Excel 中的常州、2F-01 和 2F-02。

## 查看方式

在本目录启动任意静态服务器，例如：

```powershell
python -m http.server 8766
```

打开 `http://127.0.0.1:8766/`。

## 页面状态

- `?screen=projects`：项目库
- `?screen=summary`：项目/楼层需求汇总
- `?screen=table`：计算总表
- `?screen=table&drawer=1`：计算总表与房间编辑侧栏
- `?screen=import`：收资导入预览

右下角 Tweaks 可切换舒展、标准、紧凑三档数据密度，以及三套低成本视觉主题。

## 设计意图

- 保留表三“参数为行、房间为列”的认知结构。
- 通过冻结参数列、楼层/区域筛选和分组折叠解决 200 房间的信息密度。
- 计算总表保持只读，输入通过房间侧栏完成。
- 汇总页只对可累加指标求和，非累加参数展示范围或分布。
- 颜色用于表达输入、计算结果、人工覆盖和未保存状态，不承担装饰作用。
